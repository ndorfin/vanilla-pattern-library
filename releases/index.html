A log of all changes during each publication event of the pattern library and the web assets

Most recent releases

  • Date
    Tue, 25 Jun 2024 00:00:00 GMT
    Versions
    Pattern library: 1.9.0
    Web Assets: 1.0.0
    Description

    Breaking changes. Bugfixes and improvements for cards in layouts.

    Patterns affected
    • card

      Each card's contents are now broken up into at least 4 distinct owners or rows:

      • A banner image: <picture>
      • A heading: <hX class="heading">
      • A content block: <div class="content">
      • An optional Call to Action block: <div class="call-to-action">

      Cards now determine equal heights for each of these rows when in a grid layout.

      Also added more docs to cards.

    • layout

      Layouts now create a subgrid when their children are cards.

      It is no longer possible to use cards in a list. Cards have to be direct descendents of the <div class="layout"> element.

      Added more docs to layouts.
